Eigen Decomposition

This module contains the Eigen struct which provides access to eigen decomposition functions. The motivation behind this structure is to group these algorithms behind a thin access point.

The Eigen struct provides access to the actual algorithms which do work on the underlying matrix. When calling the eigen() function on a Matrix no actual computation takes place. Because of this the Eigen struct has the must_use attribute.

Examples

use rulinalg::matrix::Matrix;

let a = Matrix::new(2, 2, vec![1.0, -2.0, -3.0, 0.5]);
println!("{:?}", a.eigen().values().unwrap());
